# Análise de Dados na Prática

## Sobre o Projeto

Este projeto foi desenvolvido durante o treinamento **Análise de Dados na Prática**, promovido pelo :contentReference[oaicite:0]{index=0}.

O objetivo do treinamento foi apresentar conceitos e técnicas de análise de dados, permitindo a aplicação prática de conhecimentos relacionados à coleta, tratamento, exploração e interpretação de dados para apoio à tomada de decisões.

## Objetivos

- Compreender os fundamentos da análise de dados.
- Realizar tratamento e limpeza de dados.
- Explorar conjuntos de dados utilizando ferramentas de análise.
- Gerar insights a partir das informações analisadas.
- Desenvolver habilidades práticas para projetos de dados.

## Tecnologias Utilizadas

- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
